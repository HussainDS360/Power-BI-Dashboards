# 📊 Data Jobs Dashboard w/ Power BI

An interactive Power BI dashboard built to analyze the 2024 data job market, including salaries, job demand, remote opportunities, and skill requirements.

![alt text](/images/Project1_Page1.png)

---

## 📖 Introduction

This dashboard was created for **Job Seekers, Career Changers, and Data Professionals** who want to better understand the current data job market.

Using a real-world dataset of 2024 data job postings, this project transforms raw job listing data into meaningful insights. Users can explore salary trends, identify in-demand skills, analyze remote work opportunities, and compare different data-related roles through an interactive Power BI experience.

---

### 💼 Business Intelligence
